Acerca del Proyecto

Este proyecto tiene como objetivo proporcionar una herramienta de búsqueda y visualización de datos para facilitar la exploración y el análisis de artículos científicos. Utilizando una base de datos en formato CSV que contiene información de artículos de diversas fuentes, como Web of Science y Scopus, la herramienta permite a los usuarios buscar artículos por palabras clave o títulos y visualizar datos relevantes a través de gráficos y visualizaciones interactivas.

La implementación se basa en tecnologías web estándar, como HTML, CSS y JavaScript, y utiliza bibliotecas como D3.js, Chart.js y PapaParse para generar visualizaciones y analizar el archivo CSV.

Este proyecto es de código abierto y está diseñado para ser fácilmente adaptable y ampliable.
